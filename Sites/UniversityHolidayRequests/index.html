Welcome to the LJMU Staff Holiday Request System

Hello and welcome to our brand new Holiday Request system. We have created this new system to make it easier for you to request your holidays.

We have created the system so that you can use your normal Username & Password which you use to access Blackboard/Email etc.

Once logged in you will be directed to the Booking Index page which will list all of the holiday requests sorted by date order.

If you have any problems with the website or the system then please don't hesitate to contact us at holidayadmin@ljmu.ac.uk
